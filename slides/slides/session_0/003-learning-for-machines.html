Machine Learning

Aprender através de exemplos

Um modelo é treinado com um conjunto de dados de exemplo. Para cada exemplo é dado o conjunto de dados de entrada e a saída esperada.

Aprender através de padrões

Um modelo é treinado com um conjunto de dados de entrada, sem saída esperada. O resultado são os padrões ou estruturas que o modelo encontra nos dados.

Aprender através da experiência

O nosso modelo aprende através da sua interação com o ambiente. A cada momento ele escolhe que acção tomar e é apropriadamente recompensado (ou penalizado) com base no resultado da sua acção.
